# GuessLoL - A League of Legends Guessing Game

GuessLoL is a League of Legends champion guessing game built to explore real-time communication using WebSockets with Java and Spring Boot, where players join interactive chat rooms to guess champions together. The backend is implemented in Java 17 with Spring Boot and Spring Boot WebSockets, the frontend uses HTML/CSS/JavaScript, and deployment was prepared for Railway.
